# Website Component Block Key

This document provides a key to the main component blocks used across the website, along with their designated names for easy reference.

- **Block A**: Hero with headline & 2 buttons
- **Block B**: Benefits grid (6 rounded cards)
- **Block C**: Testimonials strip
- **Block D**: Vertical timeline
- **Block E**: Three option cards
- **Block F**: Three use-case cards
- **Block G**: CTA banner
- **Block H**: Resources teaser
- **Block I**: Long phase cards
- **Block J**: “Why choose us” 3 icons row
- **Block K**: Contact & map layout
